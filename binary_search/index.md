# Binary Search

## Introduction
1. Two principles
   1. Shrink the search space every iteration(or recursion).
   2. Cannot exclude potential answers during each shrinking.
2. Three templates
   1. Search for a precise value.
   2. Search for a fuzzy value.
   3. universal template.
